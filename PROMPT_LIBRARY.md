## INTEGRATED ARTICLE CREATION PROMPT (Write + Edit)

You will create a publication-ready article for OneFoundr.com using 
a two-phase process: WRITE (V2.4.1 System) then EDIT (6-Pass Review).

This ensures every article is both well-written AND rigorously reviewed 
before publication.

STEP 1B: Write the Article

Using your selected elements from Step 1A, write the complete article 
following ALL V2.4.1 principles:

CORE WRITING PRINCIPLES:
□ NO forbidden phrases (check permanent + dynamic lists)
□ Vary sentence length dramatically (1-word to 30+ word sentences)
□ Use 5+ human variability elements
□ Use 3-5 different conversational devices
□ Mix paragraph lengths (1-line to 5-line paragraphs)
□ Use section header pattern library (max 2x same pattern)
□ Include credibility source naturally in content
□ Incorporate ALL research data (stats, tools, examples)
□ Write in OneFoundr voice:
  - Professional but approachable (not corporate)
  - Action-oriented (not theoretical)
  - Honest about challenges (not toxic positivity)
  - "Fellow founder" tone (not preachy expert)

STRUCTURE EXECUTION:
□ Execute selected opening type effectively
□ Follow selected structure type throughout
□ Maintain selected tone consistently
□ End with selected ending type

CONTENT REQUIREMENTS:
□ Every claim must be supported (data/example/logic)
□ All statistics must be from 2024-2025 sources
□ All tool prices must be current and accurate
□ Include 2-3 specific, concrete examples
□ Provide actionable takeaways (not just information)
